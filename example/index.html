vue2-swiper Document

Default

The direction value defaults to 'horizontal'

Demo：

page {{n}}

Code:

      
<swiper>
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>

direction

The swipers Switching direction,default is 'horizontal'.

Demo：

Set direction to 'vertical'

page {{n}}

Code：

      
<swiper direction="vertical">
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>

loop

Set true to enable the loop mode

Demo：

page {{n}}

Code：

      
<swiper loop>
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>

paginationVisible & paginationClickable

Demo：

Set paginationVisible and paginationClickable to true

page {{n}}

Code：

  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>

speed

Demo：

</swiper>

forbiddenSlide

If true, you will not change the swiper by slide

Demo：

page {{n}}

Code：

      
<swiper pagination-visible pagination-clickable forbidden-slide>
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>

tabMode & tabModeAnimation

The tabMode should be an Array, if you set tabMode , it will add a tabnav above the swiper-slide.For better effect , you should also set forbidden-slide to true and set direction to horizontal. If you just use the tab as a nav, you can set tabModeAnimation to true.

Demo：

page {{n}}

Code：

      
<swiper :tab-mode = "tabs" forbidden-slide>
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>
      
      
tabs: ['first','second','third']

Demo：

just use the tab as a nav

page {{n}}

Code：

      
<swiper :tab-mode = "tabs" tab-mode-animation>
  <div v-for="n in 3" class="random-bg">page {{n}}</div>
</swiper>
      
      
tabs: ['first','second','third']
